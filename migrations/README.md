# PostgreSQL migration'ları (AWS)

**Güncel:** Şubat 2026 – Veritabanı artık **AWS üzerindeki PostgreSQL**; Supabase kullanılmıyor.

Migration dosyalarını **sırayla** AWS PostgreSQL’de çalıştırın. Sıra ve atlanacaklar: `docs/plans/AWS_ORTAM_SIFIRDAN_KURULUM_REHBERI.md`.

- Eski konum: `supabase/migrations/` (kaldırıldı).

Klasörü taşımadıysanız: `supabase/migrations` içeriğini bu `migrations/` klasörüne kopyalayıp `supabase` klasörünü silebilirsiniz.
