# Hero "Your Child, The Hero" – Sahne Prompt Örnekleri

**Kullanım:** `scripts/generate-hero-transformation.ts` ile `--prompt="..."` argümanında kullanılır.  
**Örnek:** `--prompt="In a magical forest, holding a compass and map."`  
**`--output-name` önerileri:** `arya-forest`, `arya-space`, `arya-ocean`, `arya-castle`, `arya-pirate`, `arya-snow`, `arya-dino`, `arya-candy`, `arya-robot` vb.
